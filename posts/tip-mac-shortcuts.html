Mac keyboard shortcuts I use every single day

I resisted learning keyboard shortcuts for years. The mouse felt fine. Then I timed myself doing a few common tasks and realized how much time I was wasting moving my hand back and forth. Here are the ones that stuck.

navigation

  • Cmd + Tab — Switch between open apps. Hold Cmd and tap Tab to cycle. Let go to land.
  • Cmd + ` (backtick) — Switch between windows of the same app. Underrated.
  • Cmd + Space — Spotlight. I use this to open every app, do quick math, and search files.
  • Ctrl + Left/Right Arrow — Move between Spaces (virtual desktops).
  • Cmd + H — Hide the current app. Not Minimize — Hide. The window disappears entirely but is instantly recoverable.

text editing

  • Option + Left/Right — Move word by word instead of character by character.
  • Cmd + Left/Right — Jump to the beginning or end of the line.
  • Option + Backspace — Delete the entire word to the left of the cursor.
  • Cmd + Backspace — Delete from the cursor to the start of the line.
  • Fn + Up/Down — Page Up / Page Down.

browser (Safari / Chrome)

  • Cmd + L — Jump straight to the address bar.
  • Cmd + T — New tab.
  • Cmd + W — Close current tab.
  • Cmd + Shift + T — Reopen the last closed tab. Life-saver.
  • Cmd + Option + Left/Right — Move between tabs.

screenshots

  • Cmd + Shift + 3 — Full screenshot saved to desktop.
  • Cmd + Shift + 4 — Click-drag to capture a specific area.
  • Cmd + Shift + 4, then Space — Click on any window to capture just that window cleanly.
  • Add Ctrl to any of the above — Copies to clipboard instead of saving a file.

the one I wish I'd learned sooner

Cmd + Option + Esc — Force Quit. Brings up the Force Quit window when an app is frozen. I used to go hunting for it in the Apple menu every time.
